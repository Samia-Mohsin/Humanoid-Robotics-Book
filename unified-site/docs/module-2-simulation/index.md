# Module 2: The Digital Twin (Gazebo & Unity)

## Focus
Physics simulation and environment building.

## Learning Objectives
- Simulate physics, gravity, and collisions in Gazebo
- Create high-fidelity rendering and human-robot interaction in Unity
- Simulate sensors: LiDAR, Depth Cameras, and IMUs
- Master Gazebo simulation environment setup
- Work with URDF and SDF robot description formats

## Topics Covered
- Gazebo simulation environment setup
- URDF and SDF robot description formats
- Physics simulation and sensor simulation
- Introduction to Unity for robot visualization
- Sensor simulation: LiDAR, Depth Cameras, and IMUs
- Human-robot interaction in simulation environments

## Weekly Schedule
Weeks 6-7: Robot Simulation with Gazebo
- Gazebo simulation environment setup

- Physics simulation and sensor simulation
- Introduction to Unity for robot visualization

## Prerequisites
- Module 1: ROS 2 Fundamentals
- Basic understanding of physics concepts
